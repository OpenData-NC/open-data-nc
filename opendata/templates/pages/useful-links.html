Useful Tools & Links

{% scribble "links-page" %}

This page is intended to help you either use data or find additional data sets not included in Open-NC.org.

Bring Open N.C. to Your State

Open-NC.org is a free and open-source Django Web app. You can fork the code on GitHub at https://github.com/openrural/open-data-nc

Free, Open Source Software

  • OpenOffice – for Mac, PC and Linux. Can work with Excel files, CSV and other delimited text files.
  • R – statistical analysis software.

Free, but Not Open Source Software

  • ArcGIS Explorer Desktop – for viewing .shp files
  • Tableau Public – for PC only, a free but limited version of Tableau, a tool for creating quick data visualizations such as maps and charts

Paid Proprietary Software

  • Access
  • Oracle
  • ESRI ArcGIS for Desktop - available at little or no cost for non-profit organizations. Freely available to UNC students and faculty via ITS.

Other Catalogs of Public Data

  • Data.gov
  • NCOpenbook.com
  • GIS Library at N.C. State University
  • Investigative Reporters & Editor’s Census.ire.org

Tools for Acquiring Data

  • Freedom of Information Act letter generator - from Reporter’s Committee for Freedom of the Press.
  • State Open Records Law Request letter generator - from the Student Press Law Center.
  • MuckRock.com - a web-based serviced to help you generate and track public records requests.
  • FOIAmachine.org - a web-based serviced to help you generate and track public records requests.
  • OutWit – a downloadable program for the Mac and PC that helps scrape data off of HTML webpages. Available in a free or paid full-feature version.
  • ScreenScraper – a downloadable program for Mac, Windows or Linux that helps scrape data off of HTML webpages. Available in a free or paid full-feature version.
  • ScraperWiki – a self-service web platform where you can create and host web scrapping applications that you write in Python, Ruby, PHP or another programming language.
