Share Public Data

{% scribble "share-page-1" %}

We realize that we haven't listed every North Carolina-based government data set in the Open N.C. catalog. If you have a specific data set that you've found and would like to share, let us know by completing and submitting the form below. We'll review the information and determine whether the data set should be included in Open N.C.. The data must be specific to North Carolina, available online, and contain information in columns and rows. Our goal is to catalog government datasets that adhere as closely as possible to the 8 Principles of Open Government Data. A full list of review criteria is available on the FAQ page.

This form is for recommending specific data sets, applications, or APIs. If you'd like to nominate a general area where you think data should be more easily available, visit the Request Data page.

If the data is not already online, there are several ways you can help put it online. Remember, though, that the more disconnected data becomes from its original source the less it adheres to the 8 Principles of Open Government Data. Data sets that aren't as close to the source as possible and for which the acquisition and publication processes aren't transparent will not be included in the catalog.

Tools for Sharing Public Data


Socrata

If you have a single file in the .csv, .tsv, .xls, or .xlsx formats, you can upload it to Socrata, where users will be able to download it or access it with an API. Note that there's a 200MB limit to the size of the file you can upload. Sign up for Socrata Upload a file Upload data via the API Once you have your data in Socrata, return to this page and paste the URL in the form above.

OutWit

OutWit is an application called a scraper that you download to your PC or Macintosh computer. It helps you "scrape" data from government websites when they don't make it available in a fielded file format. After scraping a website, you will have a data file on your local computer that you would need to upload to a site such as Socrata. Download OutWit (free or paid versions available) Write your first scraper Scraping for Journalists

Scraper
Wiki

If you know a programming language such as Python or Ruby, you can use it to write very complex scrapers and make the data available via API on ScraperWiki. ScraperWiki offers a limited free service as well as robust professional subscription plans. Scraping for Journalists ebook by Paul Bradshaw ScraperWiki Tutorials ProPublica's "Scraping for Journalists"

Document
Cloud

In cases where important public information isn't recorded or available in a structured data format, journalists can use DocumentCloud to upload PDFs and then annotate and publish them. Once documents are online, they can later be turned into data using a variety of techniques and tools. DocumentCloud is a project of an organization called Investigative Reporters and Editors, and use of the site requires their permission. Sign up for DocumentCloud

DocHive

Founded by the non-profit Raleigh Public Record, DocHive is an open-source Java application that takes PDF versions of paper forms and turns them into structured data. The result is a CSV file that can be uploaded and shared via Socrata. Download DocHive

MuckRock

Many public records remain offline and available only by writing a public records request. Public records requests are a very specific type of request governed by North Carolina state law. And requests don't always yield the results you want or on a reasonable deadline. MuckRock's online service helps you track your records requests and make the disclosure process more transparent. Once digital public data are made available to you either online or on a CD-ROM, you may still need to use one or more of the other tools to share them here. Register for MuckRock FAQ on North Carolina public records law, from the N.C. Open Government Coalition North Carolina Media Law Handbook (PDF), by Cathy Packer, Hugh Stevens and C. Amanda Martin.
